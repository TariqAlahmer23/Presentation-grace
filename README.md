# GRACE Presentation Website

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Stack

- Next.js App Router
- TypeScript
- TailwindCSS
- Framer Motion
- `next/font`

## Notes

- Main page: `app/page.tsx`
- Global layout: `app/layout.tsx`
- Shared UI: `components/*`
- Theme tokens: `styles/globals.css` and `tailwind.config.ts`
- Replace the fallback logo by adding `public/grace-logo.png`
